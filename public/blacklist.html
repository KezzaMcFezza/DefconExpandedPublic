This is the leaderboard blacklist and what it does is prevent people from getting added to the leaderboard when their game gets uploaded to the website, and the only reason to use the blacklist are as follows...

If somebody submits a user request to get added to the leaderboard blacklist/to not be added to the leaderboard anymore, or if somebody just asks to be added to the blacklist

If somebody wants to be removed from the blacklist to have their name show up on the leaderboard through a user request or if somebody just asks generally through discord or game chat

If somebody wants to be added back to the leaderboard and they want all their old game stats to be added to the leaderboard, goto the leaderboard managment and add these in manually
