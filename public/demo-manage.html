So to upload a demo you need to follow these instructions or else the website wont be able to upload the demo successfully

First get the .drec file, it can be any dcrec file that you have and want to add to the demo page then you need to change the name to a name format that is easy for us to understand as this is how the dedcon servers produce them. The .dcrec file should look something like this...

game_2024-09-01-14-51.dcrec

As you can see the naming format is a date and time stamp, which to be honest isnt necessary but it really does go a long way for database integrity because down the line if every demo file has the same naming scheme it makes life alot easier. Now for the json file...

game_2024-09-01-14-58_full.json

Just for context the json file is usually around 5 to 20 minutes ahead of the .dcrec file but the website doesnt care its just easier for database managment, because the last thing i want is for the database to get corrupted and the .dcrec files and the .json files have weird naming formats

After uploading the .dcrec and the .json file the website will do the rest it will automatically add player stats to the demo page and update the leaderboard! Unless one of the players are blacklisted from the leaderboard

In conclusion its really that simple but please abide by the naming scheme so its easier to read on the database :)
I have included sample json files for uploading the games, dont be afraid they are extremely easy to understand
