If you get a virus detected popup and your browser refuses to let you download any of these files, turn off Windows Defender or any other anti-virus software as
these windows builds aren't signed so they will flag up as a virus. Also check out the Installation Tutorial and the recent and historical Patch Notes

Special thanks to (bert_the_turtle) for helping with the MacOS and Linux bundles, without his help none of this could of been possible :)

The linux build has been updated into a cross-distro bundle that will work no matter what version of linux you are running. Alongside the new addition of the MacOS bundles this means that everyone can play these modded builds regardless of your platform!

Arm64 Builds are coming very soon, but using rossetta on MacOS currently works for the x86 bundles. So you can still play if you are using a new mac!

Feel free to check the MacOS builds out and report any bugs or issues that you might encounter since its fresh out of development.

But for everyone else try out the windows installer if you are struggling with the manual install!
